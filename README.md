## ✨ תכונות עיקריות

### 👤 ממשק משתמש
- צפייה בפרטי העסק
- קביעת סיור לפי:
  - סוג שירות
  - תאריך ושעה
  - שם וטלפון
  - הערה למנהל

### 🛠️ ממשק ניהול (מוגן בסיסמה)
- `/admin` – רק למנהל
- ניהול שירותים: הוספה, עריכה ומחיקה
- צפייה ועריכה או מחיקה בכל הפגישות
- סינון לפי תאריך או שם לקוח
- צפייה ברשימת הלקוחות
